
# Hanabi
HTML5 Game

### Structures

#### Chosen Development Tools and Frameworks
* Ampersand
* Sass
* Gulp
* Lodash
* Bows
* Browser Sync

#### Chosen Gulp Modules
* AutoPrefixer
* Browserify
* JSHint
* Load Plugins
* CSS Minify
* JS Uglify
* Watch

---

### Prerequisites
* Node.js and NPM
* gulp NPM module


### First Build
* Execute `npm install`
* Execute `gulp` which will perform 'build' and starts 'watch' and 'server' tasks.


### Development Build
* Execute `gulp build`

### Production Build
* Execute `gulp build-production`
