
# Hanabi
HTML5 Game ([demo](http://rockacola.github.io/hanabi))

---

## Build

### Prerequisites
* Node.js and NPM
* gulp NPM module

### First Build
* Execute `npm install`
* Execute `gulp` which will perform 'build' and starts 'watch' and 'server' tasks.

### Development Build
* Execute `gulp build`

### Production Build
* Execute `gulp build-production`
